What is HTML DOM ?

  • HTML DOM is a standard model and programming interface for HTML.

  • HTML Codes are considered as Objects in Browser.

  • Browser copies all HTML Elements in a webpage and converts into Objects.

HTML DOM Defines

  • The HTML elements as Objects.

  • The Properties of all MITHL elements.

  • The Methods of all MITHL elements.

  • The Events for all HMML elements.
In Other Words
The HTML DOM is a standard for how to get, change, add ,or delete HTML Elements.

Architecture of Web page


This diagram appears to outline the architecture of a web browser. Hereâ€™s a breakdown of each component:

1. User Interface:

       The part of the browser that users interact with, like the address bar, back/forward buttons, and bookmarks.

2. Browser Engine:

       Acts as a bridge between the User Interface and the Rendering Engine, directing actions from the user interface to the correct functionality.

3. Rendering Engine:

      Responsible for displaying the requested web content. It interprets HTML, CSS, and potentially other resources to render a webpage.

4. Networking:

      Manages network calls, like HTTP requests, and handles communication with servers to fetch resources (HTML, images, etc.).

5. JavaScript Interpreter:

      Executes JavaScript code on the webpage, enabling dynamic and interactive content.

6. UI Backend:

       Manages the basic browser interface elements, such as dropdowns, checkboxes, and windows.

7. Data Persistence:

      Handles storage mechanisms, including cookies, cache, and session storage, to retain data locally for faster access and better user experience.
